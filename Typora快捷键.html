Typora快捷键

發表於 2020-12-06 | 更新於: 2020-12-06 | 分類於 常用工具 | | 閱讀次數:
字數統計: 294 字 | 閱讀時長 ≈ 1 分钟

官方介绍：

a markdown editor,markdown reader.

为什么选择typora？

  1. 轻便简洁

  2. 即时渲染

  3. 可读性强

  4. 学习成本低

  5. free

  6. 个人习惯：能和开源笔记Joplin结合使用

与常用markdown语法区别

  1. 标题、字体等语法支持相关快捷键
  2. 支持更多的文字格式，可在偏好设置进行设置
  3. 支持更多的对齐方式，如 <center>进行居中

常用快捷键

  • 加粗 Cmd+B

  • 标题 Cmd+数字 如一级标题 Cmd+1

  • 插入链接Cmd+K

  • 代码块 Cmd+option+C

  • 表格 option + Cmd + T

  • 引用 option + Cmd + Q

  • 水平分割线：shift + Cmd + -

  • 斜体 Cmd + I

  • 删除线 control + ~

  • 注释 control + -

  • 显示/隐藏侧边栏 shift + Cmd + L

  • 大纲视图 control + Cmd + 1

  • 文档列表视图 control + Cmd +2

  • 文件树视图：control + Cmd + 3

  • 链接引用 option + Cmd + L

  • 脚注 option + Cmd + R

  • 代码高亮 “ ` ” +回车 选择语言

  • 下标 H~2~o

  • 上标 x^2^

  • ==高亮==

  • 目录（TOC）

    输入[toc]然后回车，即可创建一个“目录”。TOC从文档中提取所有标题，其内容将自动更新。
